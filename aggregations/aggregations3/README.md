# elasticsearch-examples

## aggregations3

In this example, the documents have topics and subtopics:

```json
{
  "title" : "A mighty fine document",
  "topics": [
    {
      "name" : "vegetables",
      "subtopics" : [
        { "name": "rutabaga" },
        { "name": "artichoke" }
      ]
    }
  ]
}
```
